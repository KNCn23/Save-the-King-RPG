# Save the King 👑 - Text-Based RPG

**Save the King** is a text-based adventure game developed in Python, featuring a modular code structure, turn-based combat mechanics, and immersive storytelling.

You have successfully escaped the bandits who ambushed the King, but the danger is far from over. To reach the safety of the castle, you must traverse dangerous forests, solve ancient puzzles, and defeat the Bandit Leader.

## 🌟 Features

* **🛡️ Class Selection:** Choose your path as a **Knight** (High HP) or an **Archer** (High Damage).
* **💾 Save System:** Never lose your progress with a fully functional `JSON`-based Save/Load system.
* **⚔️ Combat System:** Experience random encounters, challenging Boss fights, and strategic turn-based combat.
* **🧩 Puzzles:** Solve the encrypted chest mystery in the Archives and answer the Guards' riddles to pass.
* **💰 Economy & NPCs:** Collect gold, trade with the Merchant, and interact with dynamic Villagers (Beware: They might accuse you of being a traitor!).
* **🎲 Random Events:** Dynamic events that change with every step, including environmental hazards like falling off cliffs.
* **🗺️ Expansive Map:** Over 20 unique locations waiting to be explored.

## 🎮 How to Play

To start the game, navigate to the project directory in your terminal and run the following command:

python game.py
