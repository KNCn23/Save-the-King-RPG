# 👑 Save the King (Kralı Kurtar RPG)

[TR] Python ile geliştirilmiş, sınıf seçimi, tur tabanlı savaşlar ve JSON kayıt sistemi içeren metin tabanlı bir RPG oyunu.

[EN] A text-based RPG game developed with Python, featuring class selection, turn-based combat, and JSON save system.

## 🛠️ Tech / Teknolojiler
- Python, JSON

## 🚀 Run / Çalıştır
`python main.py`
